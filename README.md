# 動作確認
## ローカルで実行する
```
$ git clone git@github.com:silenvx/reversi.git
$ cd reversi/app
$ npm ci
$ npm run dev
$ open http://localhost:3000/reversi
```

## デプロイ
```
$ cd app
$ npm run deploy
```
